    By default, this tournament only saves in this browser. Public hosting is optional and needs a free Firebase Realtime Database project.

    Setup instructions
    1. Create a Firebase project at firebase.google.com.
    2. Create a Realtime Database in test mode, then replace the rules with the rules below.
    3. Open project settings and copy the web app config values.
    4. Paste those values into the setup fields on this page.
    5. Click Save Hosting Setup, then Publish Public.
